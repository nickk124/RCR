SWIG linux info

in example.i, the #define SWIG_FILE_WITH_INIT line inserts a macro that
specifies that the resulting (C++) file should be built as a python extension,
inserting the module init code.

COMMANDS YOU NEED:

swig -c++ -python RCRwebutils.i
g++ -std=c++11 -O2 -fPIC -c RCRwebutils.cpp RCR.cpp FunctionalForm.cpp MiscFunctions.cpp NonParametric.cpp
g++ -std=c++11 -O2 -fPIC -c RCRwebutils_wrap.cxx -I/usr/include/python2.7
for the last command, try: g++ -std=c++11 -shared RCRwebutils.o RCR.o FunctionalForm.o MiscFunctions.o NonParametric.o RCRwebutils_wrap.o -o _RCRwebutils.so

after importing the SWIG'd python library:

example: dv = DoubleVector(10) (vector of size 10)
