Code

The source code is equipped with the maximum degree of customizability for using RCR, especially with the functional form/ model fitting portion of the algorithm. The full functional RCR source code includes support for:

  • Running RCR on any custom model function with any number of independent ("x") variables and model function parameters
  • Custom prior distribution functions for any or all of the model function parameters
  • If a model function has some x, log x etc. type of variable(s) (which is iteratively updating in RCR), it can be defined by any custom function

A documentation for the usage of the source code is included within the downloadable below.
